# Amigo Secreto

Este es un proyecto interactivo en el que podras realizar el sorteo del amigo secreto


## Instalacion 

Para instalar el proyecto se necesita clonar el repositorio 

```bash
  git clone https://github.com/OscarCuso/amigo-secreto.git
  cd amigo-secreto
```
    
## Uso del proyecto

```bash
Abrir index.html
```
